# 📂 Cisco Ansible Playbooks

[English](#english) | [Español](#español) | [Português](#português)

<a name="english"></a>
### 🇺🇸 English
Infrastructure as Code (IaC) for **Cisco IOS/IOS-XE** using Ansible.
* **Playbooks:** Infrastructure Hardening, BGP Border Provisioning, VLAN Standardization.
* **Atlas Protocol:** Always run with `--check` flag first. Ensure SSH key-based access.
